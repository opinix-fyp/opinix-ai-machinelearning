## HOW TO USE GITHUB
### STEP 1: Clone the repo
use git bash with the following code

<code>git clone https://github.com/opinix-fyp/opinix-ai-machinelearning.git</code>
### STEP 2: switch to dev branch
EVERYTHING is done in the **dev branch** if you are NOT in the dev branch, you watch out.

use this code:

<code> git checkout dev </code>

### STEP 3: MAKE A VIRTUAL ENVIRONMENT
for machine learning you need ts. in gitbash, go to the machine learning directory using the <code>cd "path to machine learning opinix"</code> thing. 

and then, in that repo, make a virtual environment with:
<code>python -m venv .venv</code> | this just makes a .venv folder with all the virtual environment stuff

and then you activate it:

on gitbash <code>source .venv/Scripts/Activate</code>

to know if its activated, you'll see (.venv) before every single prompt.
